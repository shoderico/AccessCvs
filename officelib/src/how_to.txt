2) ヘッダに __declspec(dllexport) or __declspec(dllimport) の定義を追加する

例）
#include "officelib_global.h"
// Define this symbol to __declspec(dllexport) or __declspec(dllimport)
#ifndef VBIDE_EXPORT
#define VBIDE_EXPORT OFFICELIB_SHARED_EXPORT
#endif


3) Mingwの場合：
    C++11 がサポートされない
    →　enum の forward declaration がサポートされない
    →　コンパイルエラーが出る
    ので、以下の対策をする

3-1) ヘッダーhの Q_DECLARE_OPAQUE_POINTER をコメントアウト
        access.h

3-2) ソースcppに #include <enum が定義されたヘッダ> を、自身のinclude 前に追加
        access.cpp
            #include "adodb.h"
            #include "office.h"
            #include "vbide.h"



4) Access::Application に以下のコンストラクタを追加
        access.cpp

    Application(_Application *iface)
    : QAxObject()
    {
        initializeFrom(iface);
        delete iface;
    }
.
